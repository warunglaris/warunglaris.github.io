Cara Hapus Akun

Terakhir diperbarui: 21 Maret 2026

⚠️ Peringatan Penting: Menghapus akun akan menghapus SEMUA data Anda secara permanen. Tindakan ini TIDAK dapat dibatalkan. Pastikan Anda telah membackup data penting sebelum melanjutkan.

Apa yang Terjadi Saat Akun Dihapus?

Saat Anda menghapus akun, hal berikut akan terjadi:

  • Semua data barang akan dihapus permanen
  • Riwayat transaksi akan dihapus permanen
  • Data reward referral akan dihapus permanen
  • Profil warung akan dihapus permanen
  • Nomor HP dapat digunakan kembali untuk pendaftaran akun baru

Penyimpanan Lokal: Karena semua data disimpan secara lokal di perangkat Anda, menghapus akun akan menghapus data dari database lokal HP Anda. Data tidak dikirim ke server manapun.

Cara Menghapus Akun

Metode 1: Hapus Melalui Aplikasi

Buka aplikasi Warung Kasir di HP Anda

Login dengan PIN Anda jika belum login

Menuju ke menu "Akun" atau "Profil"

Scroll ke bawah dan cari opsi "Hapus Data" atau "Hapus Akun"

Konfirmasi dengan memasukkan PIN Anda

Ikuti instruksi untuk menyelesaikan penghapusan

Metode 2: Uninstall Aplikasi

Buka Pengaturan di HP Android Anda

Cari dan pilih "Aplikasi" atau "Kelola Aplikasi"

Cari dan ketuk "Warung Kasir"

Ketuk "Uninstall" atau "Hapus Aplikasi"

Konfirmasi penghapusan aplikasi

Catatan: Meng-uninstall aplikasi akan menghapus SEMUA data termasuk akun Anda. Tidak ada data yang akan tersisa di perangkat.

Sebelum Menghapus Akun

Sebelum menghapus akun, kami sarankan Anda:

  • Catat data barang penting - Simpan daftar barang beserta harga di tempat lain
  • Cek riwayat transaksi - Review dan simpan data penjualan yang penting
  • Tukar reward - Jika Anda memiliki reward referral yang belum ditukar, tukar terlebih dahulu
  • Pindahkan data - Jika akan menggunakan akun baru di HP lain, pertimbangkan untuk memindahkan data terlebih dahulu

Setelah Menghapus Akun

Mendaftar Akun Baru

Setelah menghapus akun, Anda dapat mendaftar akun baru dengan:

  • Nomor HP yang sama - Nomor HP yang digunakan akun lama dapat digunakan kembali
  • Nama warung yang berbeda atau sama - Bebas memilih nama warung baru
  • PIN baru - Membuat PIN keamanan yang baru

Informasi: Akun baru tidak akan memiliki data dari akun lama. Anda perlu memasukkan data barang dari awal.

Pertanyaan Umum (FAQ)

Apakah data saya bisa dikembalikan setelah dihapus?

Tidak. Setelah akun dihapus, semua data akan hilang permanen dan tidak dapat dikembalikan. Pastikan untuk membackup data penting sebelum menghapus akun.

Apakah nomor HP saya terblokir setelah menghapus akun?

Tidak. Nomor HP Anda dapat digunakan kembali untuk mendaftar akun baru kapan saja.

Apakah reward referral saya hilang jika akun dihapus?

Ya. Semua reward referral yang belum ditukar akan hilang saat akun dihapus. Kami sarankan untuk menukar semua reward sebelum menghapus akun.

Bagaimana jika saya ingin menggunakan Warung Kasir lagi?

Anda dapat download ulang aplikasi dari Google Play dan mendaftar akun baru. Akun baru akan bersih tanpa data dari akun sebelumnya.

Apakah data saya masih ada di server setelah akun dihapus?

Tidak. Warung Kasir menyimpan semua data secara lokal di perangkat Anda. Tidak ada data yang disimpan di server, jadi saat akun dihapus, semua data benar-benar hilang.
