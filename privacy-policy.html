Kebijakan Privasi

Terakhir diperbarui: 21 Maret 2026

Komitmen Kami: Warung Kasir berkomitmen melindungi privasi data Anda. Aplikasi ini dirancang untuk menyimpan data secara lokal di perangkat Anda dan tidak mengirim data pribadi ke server eksternal.

1. Pendahuluan

Selamat datang di Warung Kasir ("Aplikasi"). Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data Anda saat menggunakan Aplikasi.

Dengan menggunakan Aplikasi, Anda menyetujui praktik privasi yang dijelaskan dalam kebijakan ini.

2. Data yang Kami Kumpulkan

2.1 Data yang Anda Berikan

Saat mendaftar dan menggunakan Aplikasi, Anda mungkin diminta untuk memberikan:

  • Nama Warung: Nama toko/warung Anda untuk identifikasi dalam Aplikasi
  • Nomor HP: Digunakan sebagai identitas login dan untuk fitur pemulihan PIN
  • PIN Keamanan: 6 digit PIN untuk melindungi akses ke Aplikasi
  • Kode Referral: Kode referral dari pengguna lain (opsional)

2.2 Data Transaksi dan Stok

Aplikasi menyimpan data berikut secara lokal di perangkat Anda:

  • Data barang (nama, harga, stok)
  • Riwayat transaksi penjualan
  • Data keranjang belanja
  • Riwayat reward referral

2.3 Data Lokasi

Lokasi Opsional: Aplikasi meminta izin lokasi HANYA saat pendaftaran dan bersifat opsional. Anda dapat melewati langkah ini dan tetap menggunakan Aplikasi sepenuhnya.

Jika Anda memberikan izin lokasi:

  • Data lokasi hanya digunakan untuk statistik distribusi pengguna
  • Lokasi tidak dikirim ke server eksternal
  • Lokasi disimpan sementara di database lokal

3. Cara Kami Menggunakan Data

Jenis Data Tujuan Penggunaan
Nama Warung & Nomor HP Identitas login, profil pengguna
PIN Keamanan dan proteksi akun
Data Barang & Transaksi Fungsi utama aplikasi (kasir, stok, riwayat)
Kode Referral Program referral dan reward
Nomor HP (WhatsApp) Lupa PIN, tukar reward, share data

4. Penyimpanan Data

4.1 Penyimpanan Lokal

Penyimpanan Lokal: SEMUA data Anda disimpan secara lokal di perangkat Android menggunakan database SQLite. Data TIDAK dikirim ke server cloud atau pihak ketiga.

Implikasi penyimpanan lokal:

  • Data Anda tetap ada di perangkat Anda
  • Aplikasi dapat berjalan tanpa koneksi internet
  • Kami tidak memiliki akses ke data transaksi atau stok Anda
  • Jika aplikasi di-uninstall, semua data akan hilang

4.2 Backup Data

Saat ini, Aplikasi belum menyediakan fitur backup otomatis. Kami menyarankan:

  • Catat data penting secara berkala
  • Gunakan fitur export jika tersedia di versi mendatang

5. Izin (Permissions) yang Diminta

Aplikasi meminta izin berikut dari perangkat Android Anda:

Izin Tujuan Wajib?
VIBRATE Getar saat menekan tombol untuk UX yang lebih baik Wajib
ACCESS_FINE_LOCATION Deteksi lokasi saat pendaftaran (opsional) Opsional
ACCESS_COARSE_LOCATION Deteksi lokasi kasar saat pendaftaran (opsional) Opsional

Penting: Izin lokasi DAPAT ditolak. Aplikasi akan tetap berfungsi penuh tanpa izin lokasi.

6. Pembagian Data ke Pihak Ketiga

Kami TIDAK menjual data Anda: Kami tidak menjual, menyewakan, atau membagikan data pribadi Anda kepada pihak ketiga untuk tujuan pemasaran.

6.1 WhatsApp (Fitur Share)

Aplikasi memiliki fitur untuk membuka WhatsApp dengan tujuan:

  • Lupa PIN: Mengirim PIN yang tersimpan ke WhatsApp admin
  • Tukar Reward: Mengirim data reward ke WhatsApp admin
  • Share Referral: Membagikan kode referral melalui WhatsApp

Data yang dikirim melalui WhatsApp:

  • Nomor HP Anda (sebagai pengirim)
  • PIN (hanya untuk fitur lupa PIN)
  • Data reward (hanya untuk fitur tukar reward)

6.2 Layanan Pihak Ketiga

Aplikasi menggunakan library open source berikut:

  • Flutter SDK: Framework aplikasi (Google)
  • SQFlite: Database lokal
  • URL Launcher: Membuka WhatsApp
  • Geolocator: Deteksi lokasi
  • Share Plus: Berbagi konten

7. Keamanan Data

Kami mengambil langkah-langkah berikut untuk melindungi data Anda:

  • PIN Enkripsi: PIN disimpan dalam bentuk terenkripsi di database lokal
  • Akses Lokal: Data hanya dapat diakses dari perangkat yang sama
  • Tidak Ada Server: Tidak ada server eksternal yang dapat diretas

Tanggung Jawab Pengguna: Anda bertanggung jawab untuk menjaga kerahasiaan PIN Anda. Jangan bagikan PIN kepada orang lain.

8. Hak Anda atas Data

Anda memiliki hak-hak berikut terkait data Anda:

8.1 Akses Data

Anda dapat melihat dan mengedit semua data Anda langsung melalui Aplikasi:

  • Data barang dapat dilihat di menu "Barang"
  • Riwayat transaksi di menu "Riwayat"
  • Profil dapat diubah di menu "Akun"

8.2 Hapus Data

Untuk menghapus semua data Anda:

  • Hapus Akun: Pilih menu "Akun" → "Hapus Data"
  • Uninstall Aplikasi: Semua data akan terhapus saat aplikasi di-uninstall

Peringatan: Data yang dihapus TIDAK dapat dikembalikan. Pastikan untuk mencatat data penting sebelum menghapus.

9. Data Anak-Anak

Aplikasi ini ditujukan untuk pengguna dewasa yang memiliki usaha warung. Kami tidak dengan sengaja mengumpulkan data dari anak di bawah 13 tahun. Jika Anda adalah orang tua dan mengetahui anak Anda menggunakan Aplikasi, hubungi kami.

10. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan diberitahukan melalui:

  • Pembaruan versi aplikasi di Play Store (jika tersedia)
  • Pemberitahuan di dalam aplikasi (untuk perubahan signifikan)
